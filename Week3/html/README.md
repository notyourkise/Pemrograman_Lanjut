# Week 3 — Optimalisasi, Aksesibilitas, SEO Dasar (Portofolio Final)

Tujuan:
- Menyempurnakan struktur semantik + aksesibilitas (skip link, heading, label).
- Menambahkan meta SEO dasar, lazy loading gambar, dan halaman detail proyek.
- Finalisasi portofolio (≥4 halaman) dan validasi HTML.

Struktur:
- `index.html` — Beranda dengan skip link, meta description, hero.
- `about.html` — Biodata + tabel skills.
- `projects.html` — Daftar proyek (link ke detail).
- `project-detail.html` — Detail salah satu proyek.
- `contact.html` — Form kontak yang benar.
- `assets/css/styles.css` — CSS kecil (responsif & aksesibilitas).
- `assets/img/` — Gambar.

Tugas Mingguan (Final):
1) Lengkapi konten nyata untuk semua halaman (teks, gambar, tautan proyek).
2) Tambahkan atribut `loading="lazy"` pada gambar non-hero.
3) Pastikan ada link "Lewati ke Konten" (skip link) yang terlihat saat fokus.
4) Isi `<title>` unik, `<meta name="description">` relevan tiap halaman.
5) Validasi di https://validator.w3.org/ hingga bebas error.

Checklist Final:
- Navigasi konsisten di semua halaman.
- Alt text deskriptif, label terhubung dengan input.
- Gambar responsif (max-width:100%).
- Tautan internal berfungsi.
